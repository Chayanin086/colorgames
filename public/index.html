Google Cloud

ใช้google cloud เพื่อสร้างinstanceเพื่อใช้virsaul machineหรือการที่ใช้แทนเครื่องserverในการdeploy web แล้วสร้างFirewallเพื่อกำหนดการใช้งานportว่าportไหนใช้ได้บ้าง และใช้service>storage>disk ในการสำรองข้อมูลinstance นอกจากนั้นinstanceต้องใช้ssh key โดยการgenจากเครื่องเพื่อที่จะremote ssh keyเข้าไปสั่งการเครื่องบนcloud

Docker

ใช้Dockerเพื่อช่วยรันโปรแกรมในสภาพแวดล้อมที่แยกออกจากระบบปฏิบัติการหลัก ช่วยในการส่งโค้ดได้เร็วขึ้นทำให้แอปพลิเคชันทำงานได้อย่างราบรื่นและประหยัดค่าใช้จ่ายมในการพัฒนานอกจากนั้นDockerยังช่วยควบคุมการทำงานของแอปพลิเคชันได้อย่างเต็มรูปแบบ
